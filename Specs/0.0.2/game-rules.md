# Game Rules 0.0.2

## Regla de esta version

`0.0.2` no introduce cambios de reglas.
La logica de dominio, validaciones de setup, reparto de roles, demo y flujo posterior al setup siguen heredadas de `0.0.1`.

## Se mantiene sin cambios

- minimo de 4 jugadores;
- limite maximo de 20 jugadores;
- nombres obligatorios para toda la mesa cargada;
- reparto total igual a la cantidad de jugadores;
- set de roles `Mafia`, `Policia`, `Doctor`, `Ciudadano`;
- demo fija de 8 jugadores con confirmacion previa;
- orden inicial como fuente para revelacion, pases y turnos nocturnos.

## Regla de control

Si durante la implementacion aparece una necesidad de cambiar cualquiera de estos puntos, ya no corresponde a `0.0.2` y requiere ampliar la spec de esta version o abrir una nueva iteracion.
